Learn how to take advantage of Large Language Models

Organizations are rushing to integrate Large Language Models (LLMs) to improve their online customer experience. Consequently, this technology exposes them to web LLM attacks that can exploit the model's access to data, APIs, or user information that an attacker cannot access directly.

The Web Security Academy has launched a new topic on this frontier, which goes into different techniques you can employ to manipulate LLMs alongside the various APIs they can access. It also contains four interactive labs that you can use to try out these new techniques.
